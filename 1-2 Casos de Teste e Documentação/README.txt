#- 📖 Visão Geral
    Essa documentação é sobre os casos de testes do desafio da Lacrei Saúde, criados na linguagem Gherkin (arquivos .feature) assim facilitando o entendimento e colaborações futuras

#- ⚙️ Configuração do Ambiente
    - Instalação do visual studio code (vscode)
        *IDE(Editor de texto) gratuita com suporte para Windows, Linux e Mac
        *(baixar o vscode): (https://code.visualstudio.com/download)
    
    *Recomendação:
        - Extensão 'Feature Syntax Highlight and Snippets (Cucumber/Gherkin)' - Opcional, ajuda para melhor visualização dos arquivos .feature
    
    *Observação
    - Não há necessidade de instalação de frameworks de automação. Os casos foram executados manualmente   

#- 🧪 Executando os Casos de Testes
    - Acesse a pasta "Requisitos 1 e 2/Features/"
    - Leia os arquivos .feature escritos no formato Gherkin
    - Cada arquivo .feature tem casos de testes correspondentes aos fluxos solicitados, todos feitos em forma manual sem a necessidade de automação

#- 📚 Estrutura da pasta de casos de teste

    ├── Features/
    │   ├── busca_profissional_saude.feature
    │   ├── cadastro_pessoa_usuario.feature
    │   ├── edicao_perfil_usuario.feature
    │   └── esquecer_senha_editar.feature
    ├── Vídeos/
    ├── Bugs-melhorias.txt
